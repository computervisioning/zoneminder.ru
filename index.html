ZoneMinder — это

полнофункциональная компьютерная система видеонаблюдения на базе свободного программного обеспечения.

Руководство по установке

С открытым исходным кодом

ZoneMinder работает на Linux, и его можно использовать бесплатно, что значительно снижает расходы для частных лиц и предприятий.

Вы контролируете свои данные

Ваши данные хранятся в вашей инфраструктуре. Никто кроме вас не имеет доступ к вашей конфиденциальной информации.

Удобное приложение

Специально для ZoneMinder создано удобное и быстрое приложение zmNinja для настройки, управления и просмотра, работающее на Android и Windows.

С чего начать?

ZoneMinder может быть установлен из готовых пакетов, созданных для большинства дистрибутивов, собран из исходного кода, а также запущен в контейнере Docker.
